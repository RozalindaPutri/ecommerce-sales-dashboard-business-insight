# ecommerce-sales-dashboard-business-insight

## Project Overview

This project presents an interactive e-commerce sales performance dashboard built using Looker Studio. The dashboard analyzes sales trends, product performance, regional purchasing patterns, payment methods, and failed transactions to generate actionable business insights and recommendations. The brand name used in this project (**PulseGear**) has been anonymized to maintain company confidentiality.

## Key Metrics
* Monthly Sales Trend (Revenue)
* Product Category Contribution (Apparel vs Sunglasses)
* Regional Purchasing Behavior
* Payment Method Distribution
* Failed / Unpaid Transaction Rate
* Potential Revenue Loss from Failed Transactions

## Dashboard Insights
* Sales increased significantly during the campaign period in April, indicating strong impact of promotions.
* Apparel is the most frequently purchased category, while sunglasses contribute the highest transaction value.
* Customers outside the capital city show higher average order quantity (4–6 items per transaction).
* Bank transfer dominates usage, but higher-value transactions tend to use payment gateway.
* 38 failed/unpaid transactions represent significant potential revenue loss.
* Virtual Account expiration is the main cause of failed payments.

## Tools Used
* Looker Studio
* CSV Dataset

## Notes
This project focuses on transforming raw sales transaction data into business insights through data cleaning, KPI definition, and interactive dashboard visualization.
No SQL or Python was used in this analysis.
